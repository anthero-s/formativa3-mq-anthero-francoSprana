# Projeto: Estrutura Responsiva com CSS

Este projeto é uma atividade FORMATIVA de CSS focado em **estilos globais** e **responsividade (Media Queries)**.

## Tecnologias Utilizadas
- HTML5
- CSS3 


## Responsividade (Media Queries)
O projeto adapta-se a diferentes telas:
1.  **Desktop (Padrão):** 3 colunas na grid.
2.  **Tablets (`max-width: 768px`):** Ajustado para 2 colunas na grid.
3.  **Smartphones (`max-width: 480px`):**
    *   Grid ajustada para 1 coluna.
    *   Header flexbox alterado para coluna (centralizado).
    *   Elemento `.dica-desktop` ocultado.

## Como usar
Basta abrir o arquivo `.html` vinculado ao CSS no seu navegador para visualizar as mudanças de layout redimensionando a tela.
